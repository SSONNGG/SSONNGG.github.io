Java部分

一、工作空间编码集设置

依次选择 window --> Preferences --> Genral --> Workspace，

在 Text file encoding 区域选择 Other 并选择 UTF-8，点击 Apply 后 OK 即可

二、字体设置

window --> Preferences --> Genra --> Appearance–> Color and Fonts

选中 Basic 中的 Text Font，点击 Edit 选择自己喜欢的字体以及字体大小即可

三、Java自动格式化代码设置

window --> Preferences --> Java --> Editor --> Save Actions

并按照下图所示选择即可

四、设置格式化代码时不格式化注释

window --> Preferences --> Java --> Code Style --> Formatter

取消勾选 Enable Javadoc comment formatting

如果勾选后设置页面底部出现如下提示

修改任意未存在的 Profile name 点击OK即可

JavaWeb部分

一、Html5设置

window --> Preferences --> Web -->JSP Files --> Editor --> Templates

选择 new JSP File (html)，点击 Edit

将以下代码复制粘贴替换掉 Pattern 中的所有代码，点击OK，保存即可

<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
</head>
<body>
</body>
</html>

二、服务器设置

点击 Add，选择自己已经安装好的 Tomcat 服务器版本

点击 Next，在 Tomcat installation directory 点击 Browse ，选择已安装 Tomcat 的文件夹

三、Tomcat设置

部署好 Tomcat 之后，在 Ecplise 下方 Overview 点击 Server，双击 Tomcat 服务

在弹出的界面中检查 Server Path 是否为系统安装 Tomcat 的地址

并将下方 Deploy path 部分改为 webapps（如下图）

按 Ctrl＋S 保存

之后在 Overview --> Server 中右击 Tomcat 选择 Start 启动服务器

在 Console 控制台出现如下命令服务器即启动成功

Server startup in 4101 ms

最后打开浏览器，在地址栏输入

http://localhost:8080

如果出现 Tomcat 页面则表示服务器部署成功
