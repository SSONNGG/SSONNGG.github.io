  有关于编写 Html 静态页面时如何导入自定义字体文件以更换默认字体的方法，字体文件请自行寻找，字体转换工具请自行寻找。

  在开发时请务必注意区分使用的字体是否可进行商用

一、准备文件

  首先我们需要准备到字体文件如下：

  如何获得以上文件请自行查找，网上下载的字体文件大都是 .tff 格式的文件，我们可以查找并使用字体转换工具对其进行转换成其他格式

二、建立字体CSS文件

  在我们需要导入字体的 html 的 css 文件中，创建名为 font 的文件夹，将以上文件复制进文件夹，退回到上一级 css 中，创建 css 文件 ：fontset.css

  在文件中输入以下代码

@font-face {font-family: "字体名称";
    src: url('font/字体文件.eot'); /* IE9*/
    src: url('font/字体文件.eot') format('embedded-opentype'), /* IE6-IE8 */
    url('font/字体文件.woff') format('woff'),
    url('font/字体文件.ttf') format('truetype'), /* chrome, firefox, opera, Safari, Android, iOS 4.2+*/
    url('font/字体文件.svg') format('svg'); /* iOS 4.1- */
  }

 接着在 Html 头部引用 fontset.css文件

<link rel="stylesheet" href="css/fontset.css" />

  之后我们便可以在 Html 中的 style 标签中使用font-family: "字体名称"引入自定义字体了

  如果我们想要在 Html 中直接默认使用我们导入的字体文件，我们可以在 fontset.css 中的 @front-face 下方加入如下代码设置全局字体为我们定义的字体文件。

* {
    font-family:"字体名称";
  }

  如果我们想要在一个静态页面采用多个字体文件，我们可以在上一个 @front-face 中加入其它的 @front-face 模块，改变字体路径和字体名称即可
