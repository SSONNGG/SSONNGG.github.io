215.数组中的第K个最大元素

【题目描述】

在未排序的数组中找到第 k 个最大的元素。请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

示例 1：

输入：[3,2,1,5,6,4] 和 k = 2
输出: 5

示例 2：

输入: [3,2,3,1,2,4,5,5,6] 和 k = 4
输出: 4

说明:

你可以假设 k 总是有效的，且 1 ≤ k ≤ 数组的长度。

【题解】

快速排序方法：

假定一个数为轴值，这里直接选定最左边的数作为轴值

[3,2,1,5,6,4]和 k = 2
选取轴值为：3

此时将3与后边的数进行比较，分成两个部分：

左边为比轴值小的部分

右边为比轴值大的部分

[2,1,3,5,6,4]

由于 k =2 ，由题意可知，我们需要第2大的元素

也就是说我们需要排序好的数组中倒数第二个数，也就是数组下标为 size-k 的数（注意数组是从0开始的），本题中为数组下标为4，也就是数组中第5个数。

此时对轴值3右边的数组为：

[2,1,3 | 5,6,4]

同样选取最左边的数5为轴值，将其与后边的数字进行比较

将小于轴值的数放在轴值左边，将大于轴值的数放在轴值右边

有：

[2,1,3 | 4,5,6]

此时我们可以返回出数组中下标为 size-k 的数，即为结果

[2,1,3,4,5,6]
返回：5

代码实现：

import java.util.Random;
class Solution{
    int [] nums;
    //辅助函数，用于交换
    public void swap(int a,int b){
        int tmp=this.nums[a];
        this.nums[a] = this.nums[b];
        this.nums[b] = tmp;
    }

    //把比轴值小的数放在轴值左边，比轴值打的数放在轴值右边，返回轴值下标
    public int partition(int left,int right,int pivot_index){
        //暂存轴值元素
        int pivot = this.nums[pivot_index];
        //为了方便比较，把轴值放在数组最右边
        swap(pivot_index,right);
        //初始化第一个比轴值小的位置（数组的最左边）
        int store_index = left;
        //遍历数组调整轴值的位置，一旦发现元素值小于轴值，就将元素放到左侧，此时轴值的位置右移
        for(int i=left;i<=right;i++){
            if(this.nums[i]<pivot){//当循环到的元素小于轴值时
                //数组下标交换（元素位置交换）
                swap(store_index,i);
                store_index++;//轴值右移
            }
        }
        //遍历完成后，store_index是轴值应该所在的位置，可是里边存储着大于轴值的元素，right此时存储的是轴值本身，因此要对双方进行交换
        //把轴值放到预期的位置
        swap(store_index,right);
        return store_index;
    }

    //选择轴值，并且根据调整后轴值的位置与k的相对位置关系，进行进一步搜索（递归实现）
    public int quickselect(int left,int right,int k_smallest){
        //定义递归出口
        if(left == right){
            return this.nums[left];
        }
        //随机生成轴值，也可以每次选取最左边的值作为轴值，效果是相同的
        Random random_num=new Random();
        int pivot_index=left+random_num.nextInt(right-left);
        pivot_index=partition(left,right,pivot_index);
        //根据轴值位置与K的相对关系，寻找对应的半边
        if(k_smallest==pivot_index)	//当轴值位置等于k时，直接返回这个值
            return this.nums[k_smallest];
        else if(k_smallest<pivot_index)	//当k小于轴值，从数组左边寻找
            return quickselect(left,pivot_index-1,k_smallest);
        return quickselect(pivot_index+1,right,k_smallest);//当k大于轴值，从数组右边寻找
    }

    public int findKthLargest(int [] nums,int k){
        this.nums=nums;
        int size=nums.length;
        //将k转换为数组下标为 size-k 的数
        return quickselect(0,size-1,size-k);
    }
}

206. 反转链表

【题目描述】

反转一个单链表。

示例：

输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL

【题解】

1.迭代

题目要求做到链表的反转，此时会有思路：

需要两个指针 prev 与 curr

最开始时：prev=null, curr=head

只需要将两个指针指向的地址也就是 curr.next 反转过来即可

1->2->3->4->5->NULL
prev=null
curr=head
curr.next=prev
NULL<-1 -> 2->3->4->5

curr.next=prev

之后将两个指针向后移动一位持续操作知道链表结束为止。

但是此时会出现一个问题：

当第一个 curr.next=prev 操作完成时，此时 curr.next 后继节点是反转后的地址了，也就是链表断开了，此时curr 与 prev 不能向前移动了。

NULL<-1 | 2->3->4->5

因此需要一个中间指针，给予一个中间节点帮助 curr 与 prev 移动到下一位

1->2->3->4->5->NULL
prev=null
curr=head
next=curr.next	中间指针 1->2
curr.next=prev	NULL<-1
prev = curr		prev向后移动
curr = next		curr向后移动
不断循环操作即可
直到curr=NULL

代码如下：

public ListNode reverseList(ListNode head) {
    //定义指针
    ListNode prev = null;
    ListNode curr = head;	//用于遍历每个节点
    while (curr != null) {
        ListNode next = curr.next;	//记住这个节点的下一个节点
        curr.next = prev;			//将节点反转
        prev = curr;				//向后移动
        curr = next;
    }
    return prev;		//执行结束后，表头在prev，因此要返回prev的值
}

2.递归

递归的流程大致如下

1->2->3->4->5->NULL

2->3->4->5->NULL

3->4->5->NULL

4->5->NULL

5->NULL

倒过来就是

5->NULL
head
4->5->NULL
head
3->4->5->NULL
head
2->3->4->5->NULL
head
1->2->3->4->5->NULL
head

如何实现反转呢？

第一次反转
head.next.next=head;
head.next=null;
也就是5->NULL
变为NULL<-NULL<-5

第二个反转
除了头结点外，剩余的其他节点在上一次反转已经排好序了：NULL<-5
所以只需要反转头结点
NULL<-5->4
head.next.next=head;
head.next=null;
NULL<-5<-4
重复递归
直到链表反转结束

代码如下：

public ListNode reverseList(ListNode head) {
    //递归出口，如果head为null或者只剩下一个元素也就是head.next==null的时候，返回
    if (head == null || head.next == null) {
        return head;
    }
    //保存每一次递归完成后的链表，执行递归
    ListNode p = reverseList(head.next);
    head.next.next = head;	//反转
    head.next = null;
    return p;	//每一次递归结束后返回的最后一个节点（反转后的头结点）
}

3. 无重复字符的最长子串

【题目描述】

给定一个字符串，请你找出其中不含有重复字符的最长子串的长度。

示例 1:

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

示例 4:

输入: s = ""
输出: 0

【题解】

滑动窗口

题目中出现：”出现字数“关键字，需要用到散列表

题目中出现：“子串”关键字，考虑滑动窗口

题目要求不出现重复子串，以“abcabcbb”为例子

使用双指针解决

前指针代表窗口的前面位置，后指针代表窗口的后位置
当字符不相同时，前指针前移一位，后指针不动
当出现相同字符时，收缩滑动窗口，收缩的大小取决于与当前字符重复的字符上一次出现的位置与当前字符的位置
收缩应该使得子串刚好排除掉重复的字符
创建一个整形max=0记录最大子串的长度
由中间变量tmp=font-back+1与max进行比较，当tmp大于max的时候，将tmp赋值给max，最终返回max

|| a b c a b c b ba b c a b c b b
a 不重复，前指针前移，max++
|a| b c a b c b b
b 不重复，前指针前移，max++
|a b| c a b c b b
c 不重复，前指针前移，max++
|a b c| a b c b b
a 重复，收缩窗口，后指针移动排除上次出现的 a
a |b c a| b c b b
b 重复，收缩窗口，后指针移动排除上次出现的 b
a b |c a b| c b b
c 重复，收缩窗口，后指针移动排除上次出现的 c
a b c |a b c| b b
b 重复，收缩窗口，后指针移动排除上次出现的 b
a b c a b |c b| b
b 重复，收缩窗口，后指针移动排除上次出现的 b
a b c a b c b |b|
前指针走到尽头，算法结束

代码如下：

class Solution {
    // 双指针，前指针遇到重复字符则移动后指针，直到前指针走到尽头
    public int lengthOfLongestSubstring(String s) {
        if (s == null || s.length() == 0) {
            return 0;
        }
        Set<Character> chars = new HashSet<Character>();
        //定义前指针和后指针的位置
        int back = 0;
        int front = 0;
        int max = 0;	//记录长度
        while(front < s.length()) {		//当前指针走到尽头，算法结束
            char currentChar = s.charAt(front);
            if (!chars.contains(currentChar)) { // 不重复则前指针继续向前
                chars.add(currentChar); //记录
                int tmpMax = front - back + 1;	
                if (tmpMax > max) {	//最大子串长度有变化，更新max
                    max = tmpMax;	
                }
            } else { //重复则后指针向前移动
                char backChar = s.charAt(back);
                while(backChar != currentChar) { //一直移动到重复字符处
                    chars.remove(backChar);
                    ++back;
                    backChar = s.charAt(back);
                }
                // 后指针越过重复字符
                ++back;
            }
            ++front; //前指针一直向前
        }
        return max;
    }
}
